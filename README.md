# 起業への軌跡 - Entrepreneurship Journey

2025年3月の退職から始まる、私の起業の記録と軌跡をまとめたリポジトリです。

## 私について

保育園栄養士として7年間勤務。2025年3月に退職し、起業の道へ。
栄養士 × デザイン × コーディングという希少な組み合わせを武器に、新しい働き方を模索中。

## ビジョン

**経営者として事業を運営しながら、自分はWebでコツコツ稼ぐ**

- M&Aで小規模事業を買収・経営
- Web制作・デザインで安定収益
- 心地よく、自分らしく働く

詳細は [`vision-history/`](./vision-history/) を参照。

## リポジトリ構成

```
startup-journey/
├── README.md          # このファイル
├── plan/              # 事業計画・スケジュール
├── vision-history/    # ビジョンの変更履歴（日付ごと）
├── daily/             # 日々の記録（月ごと1ファイル）
├── outputs/           # 学習のアウトプット・調査結果
└── milestones/        # 重要な達成事項の記録
```

## 活用方法

### 📊 事業計画 (`plan/`)
- 事業計画とスケジュールのまとめ
- vision-historyとdailyの内容を元に整理
- 月ごとに更新（例: `2025-12_plan.md`）

### 🎯 ビジョン履歴 (`vision-history/`)
- 起業プラン・事業戦略の変遷
- 日付ごとに保存（更新のたび追加）
- ファイル名: `2025-12-01_vision.md`

### 📝 日々の記録 (`daily/`)
- ふとした気づき
- その日の思考
- 感情の変化
- 小さな決断
- 学習内容と所感（月ごとに1ファイル）

### 📚 アウトプット (`outputs/`)
- 読んだ本の要約
- M&Aの調査結果
- 会計・財務の学習メモ
- 事業計画のドラフト
- Web制作の実績記録

### 🎯 マイルストーン (`milestones/`)
- 法人設立完了
- 初案件受注
- M&A案件買収
- 月次収益達成
- その他重要な節目

## 2025年のゴール

1. **法人設立完了** (3月)
2. **Web制作で月20〜40万円の収益** (6月〜)
3. **M&A案件を1件買収** (7月〜2026年3月)
4. **買収した事業を黒字で運営** (年末まで)

## タイムライン

- **2025年1月〜2月**: 勉強・準備期間
- **2025年3月末**: 保育園退職・法人設立
- **2025年4月〜**: 本格始動
- **2025年6月〜**: 収益化目標
- **2025年7月〜2026年3月**: M&A実行目標
